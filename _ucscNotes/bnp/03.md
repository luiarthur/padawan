Consider a probability space ($\Omega, \mathcal F, P$), where $\Omega$ is the sample
space of the experiment, an index set $T$, and a state space $S$. A stochastic
process is a collection
X = {X(ω, t) : ω ∈ Ω, t ∈ T}
